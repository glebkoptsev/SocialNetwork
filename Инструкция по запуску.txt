1. Перейти в расположение этого файла
2. Выполнить в командной строке команду: docker compose pull
3. Выполнить в командной строке команду: docker compose up -d
4. Импортировать в postman коллекцию First Task.postman_collection.json
5. По адресу http://localhost:5000/swagger доступен сваггер
6. Для наката структуры базы пользователей включить контейнер cache_warmup
7. В ДЗ по кешированию структура БД была изменена
8. Скрипты БД диалогов лежат в файле DialogServiceScripts.txt
